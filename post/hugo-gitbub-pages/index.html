Building a personal website with Hugo in localhost and them commit to the github papes. it was based on the Build a Personal Website With Github Pages and Hugo. In this blog, the operation system is MacOS. Following this, I test the hugo and git in the Ubuntu and Windows 10

In the WSL Ubuntu:

I want to do it in the window subsystem for linux (Ubuntu 18.04.2 LTS).

In the window 10:

To install hugo in window 10, first make folders

1
2
3

/Hugo
/Hugo/bin
/Hugo/sites

1 download hugo and set up the PATH https://github.com/gohugoio/hugo/releases select hugo_0.80.0_Windows-64bit.zip and download to /hugo/bin. extract all files and make sure hugo.exe in /hugo/bin.

2 Add the /hugo/bin to the PATH

3 hugo help test the install successfully

4 make a new site name mysite

1
2
3

mkdir website
cd website
hugo new site mysite

5 choose a theme

1
2
3
4

cd mysite
cd themes
git clone https://github.com/vjeantet/hugo-theme-casper

Here you will have a new site and now let us set up the configuration. Find config.toml in mysite folder. open it with text editor. Find the example configuration by https://github.com/vjeantet/vjeantet.fr/blob/master/config.toml.

6 create the first page

1
2
3

cd mysite
hugo new about.md
# you will find a file about.md in the content folder

7 test the page in the localhost

1
2

cd mysite
hugo server

Go to http://localhost:1313/about in your web browser and the page is shown

8 create the first blog

since the hugo server is running, you can open a new bash

1
2

cd mysite
hugo new post/first.md

edit the first.md with your text editor and then go to http://localhost:1313. You will see the index page.

You can build the site successfully but here I found that no blog list in the index when you scroll down. There is only one item post. when you clik the post, the blog list will appear.It seems this theme is too old and was parsed wrong.
